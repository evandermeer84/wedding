Typing Lab Created

For reasons of privacy and protected information, I can not post the code for this project, as it is the solution and grading scripts to a lab assignment that is currently being used by the department of computer science at the U of A. This is a simple typing game with three levels, written in Assembly to be run on the Risc-V simulator as it uses the memory mapped IO simulator and timer funcationality. This assignment was conceputalized by me, and the solution was written as a reference for future TA's. I also wrote am instructional webpage as a guide for students.
